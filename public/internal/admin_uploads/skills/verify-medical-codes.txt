name: "verify_medical_codes"
    description: "Queries official medical registries to verify standard codes (RxNorm, LOINC, MedDRA) before adding them to the database."
    tools: 
      - "search_web"
    instructions: |
      PURPOSE: Ensure the PPN Knowledge Graph relies on factual, industry-standard interoperability codes[cite: 5759, 6163].
      TRIGGER: Run this skill whenever a new substance, outcome assessment, or adverse event is added to the system.
      STEPS:
      1. Use `search_web` to query the official registry for the requested item.
         - For Substances: Search "RxNorm CUI for [Substance]".
         - For Assessments/Questionnaires: Search "LOINC code for [Assessment]".
         - For Adverse Events: Search "MedDRA code for [Event]".
      2. Extract the exact standard code. DO NOT hallucinate or guess codes.
      3. Use the verified code in the subsequent database insertion or schema update.
